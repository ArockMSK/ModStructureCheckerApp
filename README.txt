Mod Structure Checker

English
Description:
Mod Structure Checker is a Windows Forms application designed to streamline mod development for RimWorld by facilitating communication with neural networks (e.g., AI assistants like Grok). It generates a detailed report of mod folder structures, which helps neural networks analyze and troubleshoot errors in RimWorld mods efficiently.

Features:
- Select a folder containing RimWorld mods.
- Choose a save location for the output file.
- Generate a comprehensive report (ModFullData.txt) with:
  - Directory structure of the mod.
  - Contents of text files (.xml, .cs, .txt) for code analysis.
  - Metadata of image files (.png, .jpg, .jpeg, .gif, .bmp) for asset verification.
- Multi-language support: English, Russian, Chinese (Simplified).
- Automatic language detection based on system settings.
- Error logging and display in a separate window for easy debugging.

Usage:
1. Run the application.
2. Click "Select Folder" to choose a RimWorld mod directory.
3. Optionally, click "Select Save Folder" to specify where to save the report.
4. Click "Run Analysis" to generate the report.
5. Share the report with a neural network (e.g., Grok) to get help with mod errors.
6. Use the "Language" menu to switch between English, Russian, or Chinese.

System Requirements:
- Operating System: Windows 7, 8, 10, or 11 (32-bit or 64-bit).
- .NET 8.0 Runtime:
  - Required to run the application. This is not pre-installed on all systems.
  - Check your system architecture (32-bit or 64-bit):
    - For 32-bit (x86) systems: Download from https://dotnet.microsoft.com/en-us/download/dotnet/8.0/runtime-8.0.0-win-x86
    - For 64-bit (x64) systems: Download from https://dotnet.microsoft.com/en-us/download/dotnet/8.0/runtime-8.0.0-win-x64
  - Installation: Run the downloaded installer and follow the on-screen instructions.
  - Note: If you’re unsure about your system type, right-click 'This PC' or 'My Computer' → 'Properties' to check.
- Disk Space: Approximately 5-10 MB for the application, plus additional space for generated reports.
- No additional software required.

Distribution:
- The application is provided as two standalone executable files:
  - ModStructureCheckerApp.exe (for 64-bit systems, win-x64)
  - ModStructureCheckerApp.exe (for 32-bit systems, win-x86)
- Download the appropriate version based on your system architecture and ensure .NET 8.0 Runtime is installed.

Русский
Описание:
Mod Structure Checker — это приложение на Windows Forms, созданное для упрощения разработки модов для RimWorld путём улучшения взаимодействия с нейросетями (например, ИИ-помощниками вроде Grok). Программа создаёт подробный отчёт о структуре папок модов, который помогает нейросетям быстро анализировать и находить решения для ошибок в модах RimWorld.

Возможности:
- Выбор папки с модами RimWorld.
- Указание места сохранения отчёта.
- Генерация полного отчёта (ModFullData.txt) с:
  - Структурой директорий мода.
  - Содержимым текстовых файлов (.xml, .cs, .txt) для анализа кода.
  - Метаданными файлов изображений (.png, .jpg, .jpeg, .gif, .bmp) для проверки ресурсов.
- Поддержка нескольких языков: английский, русский, китайский (упрощённый).
- Автоматическое определение языка на основе системных настроек.
- Логирование ошибок и их отображение в отдельном окне для удобной отладки.

Использование:
1. Запустите приложение.
2. Нажмите "Выбрать папку" для выбора директории с модами RimWorld.
3. При необходимости нажмите "Выбрать папку сохранения" для указания места сохранения отчёта.
4. Нажмите "Запустить сборку" для генерации отчёта.
5. Передайте отчёт нейросети (например, Grok), чтобы получить помощь с ошибками мода.
6. Используйте меню "Language" для переключения между английским, русским или китайским.

中文 (简体)
描述:
Mod Structure Checker 是一款 Windows Forms 应用程序，旨在通过优化与神经网络（例如 Grok 等 AI 助手）的交互来简化《RimWorld》的模组开发。它会生成模组文件夹结构的详细报告，帮助神经网络快速分析并解决《RimWorld》模组中的错误。

功能:
- 选择包含《RimWorld》模组的文件夹。
- 指定报告的保存位置。
- 生成全面报告 (ModFullData.txt)，包含：
  - 模组的目录结构。
  - 文本文件内容 (.xml, .cs, .txt) 用于代码分析。
  - 图像文件元データ (.png, .jpg, .jpeg, .gif, .bmp) 用于资源验证。
- 支持多语言：英语、俄语、简体中文。
- 根据系统设置自动检测语言。
- 在单独窗口中记录和显示错误，便于调试。

使用方法:
1. 运行应用程序。
2. 点击“选择文件夹”以选择《RimWorld》模组目录。
3. 可选：点击“选择保存文件夹”以指定报告保存位置。
4. 点击“运行分析”以生成报告。
5. 将报告分享给神经网络（例如 Grok），以获取模组错误的解决方案。
6. 使用“Language”菜单在英语、俄语或中文之间切换。

系统要求:
- 操作系统：Windows 7、8、10 或 11（32 位或 64 位）。
- .NET 8.0 Runtime:
  - 运行应用程序所需。某些系统中可能未预装。
  - 检查您的系统架构（32 位或 64 位）：
    - 对于 32 位（x86）系统：从 https://dotnet.microsoft.com/en-us/download/dotnet/8.0/runtime-8.0.0-win-x86 下载
    - 对于 64 位（x64）系统：从 https://dotnet.microsoft.com/en-us/download/dotnet/8.0/runtime-8.0.0-win-x64 下载
  - 安装：运行下载的安装程序并按照屏幕上的说明操作。
  - 注意：如果不确定系统类型，右键点击“此电脑”→“属性”查看。
- 磁盘空间：应用程序约需 5-10 MB，外加生成的报告所需空间。
- 无需额外软件。

分发:
